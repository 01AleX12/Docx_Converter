# Word to HTML Converter

Aplicación web sencilla desarrollada en **JavaScript** para convertir documentos **Word (.docx)** en **HTML** de forma rápida y automática.

---

## Funcionalidades

- Subir un archivo Word (.docx)
- Generar código HTML limpio a partir del contenido
- Descargar el HTML generado
- Compatible con estilos básicos de Word (negrita, cursiva, listas, títulos)

---

## Tecnologías utilizadas

- **JavaScript** (ES6+)  
- **HTML5 / CSS3**  
- **FileReader API**  
- **docx.js** o librería similar para lectura de documentos Word

---

## Estructura del proyecto

- index.html → Interfaz de usuario
- styles.css → Estilos de la página
- script.js → Lógica de conversión Word → HTML

---

## Instalación y uso

1. Clonar o descargar el repositorio.
2. Abrir `index.html` en un navegador moderno.
3. Seleccionar un archivo `.docx`.
4. Visualizar el HTML generado y copiarlo o descargarlo.
